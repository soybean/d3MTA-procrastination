What is this project?
This project uses a steamgraph to show the differences in failures in the NYC MTA subway lines.
How to use this project
When the page is first loaded, all of the subway lines are showed.
You can hover over the graph to display the %failure and the given date of that point.
You can click on the subway line icon to show that subway line's data.
How was this project made?
This project was made using d3 and data from the MTA
